# Biblioteca API

API para gestión de biblioteca desarrollada con Node.js, Express, Sequelize y MySQL.

## Tecnologías

- Node.js
- Express
- Sequelize
- MySQL
- dotenv
- cors

## Instalación

npm install

## Configuración

Crear un archivo `.env` usando como base `.env.example`.

## Ejecución

npm start

## Endpoints iniciales

GET /

GET /api/libros
